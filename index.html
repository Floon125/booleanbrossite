What is this game?

Into the Origin was a process of combining the various ideas of multiple people in coherent ways, thus formulating a great game. As seen in the original method used to create a game which is both a roguelike with no out-of-game progression and a primarily story-based game, even ideas which may not make sense initially can be used together in unique ways. Between the five members known as the Boolean Brothers, the team was able to blend these elements into a single game using the Godot game engine and GDScript.
